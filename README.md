# Your Energy

Web application for browsing the exercise catalog, searching by category, and
managing favorites.

## Features

- Browse exercises by body parts, muscles, and equipment
- Search and pagination
- Persistent storage via LocalStorage
- Rate and review exercises
- Motivational quotes
- Responsive layout

## Technologies

- HTML5, CSS3
- Vanilla JavaScript (ES Modules)
- Vite
- Axios
- modern-normalize

## Структура проєкту

```
src/
  css/            # styles
  fonts/          # fonts
  img/            # static images
  js/             # application logic
  partials/       # HTML parts (injected via Vite)
  index.html
  favorites.html
```

### Start

```bash
# Install
npm install

# Development
npm run dev

# Production build
npm run build

# Preview build
npm run preview
```
